131  ## @brief Loads all structures in a single .traj file into a database.
132  #
133  # Useful for adding all structures from a minima hopping run into
134  # the database, in which case "relaxed=False" should be interpreted
135  # as structures with their original inaccurate energies and forces,
136  # while "relaxed=True" means that the energies/forces have been updated
137  # to the level of accuracy needed from the respective method.
138  #
139  # In the case that the structures are from minima hopping, eqm_struct
140  # should be set to False so these structures are identifiable.
141  #
142  # The .traj file should start with "batch_", followed by the name of the
143  # structures contained within it.
144  #
145  #
146  # @param traj_file **str**
147  # Path to ASE trajectory file containing relaxed structures. The filename should
148  # start with "batch_", followed by the name of the structures contained within it.
149  # @param relaxed **boolean**
150  # Whether the structures are in their final state or require further calculation.
151  # @param eqm_struct **boolean; Optional**
152  # Whether this structure is a ground state structure, eg. from a geometry optimisation,
153  # or a non-equilibrium structure, eg. from minima hopping.
154  #
