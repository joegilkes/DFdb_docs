Detailed Description

DFdb analysis module.

Contains the DFanalysis class for plotting comparative results, as well as the (now deprecated) ari2list function for converting lazy iterators to lists.

def DFdb.analyse.ari2list (   ari)

Deprecated Converts an AtomsRow Iterator object to a list of AtomsRow objects.

If a selection is made directly from the ASE Database object (using DFdatabase.dc.select), a lazy iterator over AtomsRows will be returned which gets destroyed after iterating over it for the first time. This iterator also cannot be enumerated over.

This can be fixed by preconverting the lazy iterator into a list. However, for large selections this can take up a large amount of memory.

Parameters
arilazy iterator (generator object)
AtomsRow iterator obtained by direct database selection.
