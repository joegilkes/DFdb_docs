Detailed Description

DFdb options module.

Contains the DFoptions class for handling global options for an instance of DFdb.DFdatabase.

Typical usage:

from DFdb.db import DFdatabase
# Not specifying options here creates a new DFoptions instance with default values.
db = DFdatabase('/path/to/db_file', 'geometry')
db.options.load('/path/to/options_file')
db.options.get()
db.options.set_globals('export_folder', '/path/to/new_export_folder')
db.options.set_calc_options('aims_xc', 'b3lyp')
db.options.save('/path/to/my_new_options_file')
