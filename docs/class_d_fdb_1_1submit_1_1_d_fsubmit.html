Detailed Description

Handles communication with various queueing systems.

Contains functions for submitting geometry relaxation jobs on single structures, as well as force/energy evaluations on batches of geometries.

Currently has support for submitting jobs on Orac (using SLURM) and Young (using their custom version of PBS). Support for Avon is planned. Also supports creating job files but not submitting, which is useful for small-scale DFTB+ calculations which don't need to run on HPC systems.

Parameters
calc_folderstr
Folder for all calculations to take place in.
optionsDFoptions object
Object to extract global/calculator options from. Usually best to use the current DFdatabase instance's options by passing DFdb.db.DFdatabase.options.
subfolderstr; Optional
Folder within calc_folder to place calculations in.
