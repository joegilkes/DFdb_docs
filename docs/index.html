DFdb Python Package

DFdb is an ASE database wrapper package, made to facilitate the creation, propagation and analysis of comparative benchmarking databases.

  • Interactive design, made for use in an iPython Jupyter notebook.
  • Wraps inputting of unrelaxed structures, relaxed structures and reactions calculated with NEB into ASE SQLite3 databases with a rich tagging system for easy filtering of database entries.
  • Contains HPC job submission methods and premade ASE calculators so new optimisations can be submitted to queues on-the-fly with minimal user interaction required.
  • Analysis routines allow the user to perform large-scale comparisons across specified slices of databases, with options to report accurate statistics from the data. Currently implemented comparisons include:
    • Least-RMSD of atomic positions
    • RMSD of atomic forces
    • Energy differences relative to a known structure

DFdb was originally built to aid in benchmarking of DFTB parameter sets against hybrid-DFT calculations. However, it could easily be modified for other benchmarking purposes.

Documentation

Documentation is extracted from the code and compiled by Doxygen on every push to the main branch. It is available here. If you are on the documentation but don't have access to the repository, email me at [j.gil.nosp@m.kes@.nosp@m.warwi.nosp@m.ck.a.nosp@m.c.uk] to request access.

DFdb comes with a full tutorial for introducing new users to its usage and guiding them through the process of reproducing results from the two siloxane databases currently provided with the package. For more information, check out notebooks/Tutorial.ipynb. The tutorial and result reproduction only require use of DFTB+, and can be run on any modern laptop/desktop computer.

Requirements

In order to function correctly, DFdb requires the following Python 3 packages:

  • NumPy
  • Matplotlib
  • Pandas
  • Seaborn
  • Scikit-learn
  • ASE

In addition, to perform DFT and DFTB calculations, MPI-aware binaries of FHI-aims and DFTB+ are required.
