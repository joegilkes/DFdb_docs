Detailed Description

Main ASE database wrapper.

Contains the DFdatabase class, which wraps ASE's db module to add additional functionality. This includes functions for adding structures to a database with additional metadata to aid in searching and filtering, as well as functions for getting rows from a database and exporting these rows back out to files.

Typical usage:

from DFdb.db import DFdatabase
db = DFdatabase('/path/to/db_file', 'geometry')
db.add_relaxed('/path/to/structure')
all_relaxed = db.select(relaxed=1)
db.export(all_relaxed, filename='all_relaxed.traj')
